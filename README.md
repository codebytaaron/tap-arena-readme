## Overview

Tap Arena Masters is an arcade-style game where players tap rapidly to compete in short arena battles. The goal is to outpace opponents through speed, timing, and consistency.

The game is designed to be easy to learn, fast to play, and visually clean.

## Features

- Tap-based competitive gameplay  
- Quick arena-style rounds  
- Simple controls  
- Clean, minimal UI  
- Smooth browser performance  

## Tech Used
